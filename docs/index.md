# Darniri Documentation

Darniri is a tiling window manager for macOS with scrolling columns, a unified command palette, and an overview mode.

This site is the documentation hub for architecture notes and technical reference material. For the repository overview and screenshots, see the [project README](https://github.com/stearz/Darniri/blob/main/README.md).
